## Dataset Description

This dataset contains weather data collected from **January 1, 2024, to December 31, 2025**, for **20 major U.S. cities**.  
The dataset includes meteorological variables such as:

- Temperature  
- Humidity  
- Wind Speed  
- Precipitation  
- Cloud Cover  
- Pressure  

The target variable is **Rain Tomorrow**, indicating whether rainfall is expected on the following day.
